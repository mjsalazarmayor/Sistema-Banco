# Sistema de Banca en Linea
Un sistema bancario administra cuentas financieras. El sistema define una clase abstracta Cuenta con numero unico de 10 digitos, titular y saldo disponible. Existen cuentas de ahorro con interés anual del 2% y cuentas corrientes con limite de sobregiro de 500.000 pesos. El sistema registra transacciones con identificador, tipo (DEPOSITO, RETIRO, TRANSFERENCIA), monto y fecha. Las transferencias entre cuentas actualizan los saldos inmediatamente y se rechazan cuando el monto excede el saldo disponible mas el limite permitido.
